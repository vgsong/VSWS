PROJECTS.

INVOICE UPLOADER

Tools: Python (pandas, selenium, pyautogui)

Background: The company used to upload hundreds of invoices manually into a client's portal. This script obtains invoice information from a csv file and python initiates selenium browser driver, logs into the portal, creates an invoice upload form, fills the form and submits it. The process got automated (not fully), which freed the time for the whole department. The BOT requires minimum setup time and mapping. Around 1600 invoices have been uploaded since August 2021.

GitHub URL

INVOICE FILER

Tools: Python (standard library)

Background: Every month invoices need to be filed in the network drive. The invoices are filed based on project/subproject and year (YYYY) and month (MM MMM YYYY) period. The bot uses an invoice mapping from csv file, and moves invoices from a repository folder into its respective program/project/subproject, and proper period (YYYY and MM MMM YYYY) directory. If the directory does not exist, the bot creates one, and moves the invoice into the just created directory.

GitHub URL

INVOICE PDF READER

Tools: Python (pandas, PyPDF2, tkinter)

Background: Hundreds of invoices are processed monthly, and each of them requires to be renamed using company's SOP and its information transposed into excel. With PyPDF2, the script extracts all the strings from the PDF invoice, filters the information using RegEx, and appends them into an excel spreadsheet. This script helps ease out the manual data entry portion, as well as the invoice renaming process. This script also supports the INVOICE BOT UPLOADER.
