INVOICE BOT UPLOADER

Tools: Python (selenium, pandas, pyautogui, PyPDF2, tkinter)

BACKGROUND/PROCEDURE:

The company used to upload hundreds of invoices manually into a client's portal. I wrote a script that obtains invoice information from a csv file and python initiates selenium browser driver, logs into the portal, creates an invoice upload form, fills the form and submits it. The process got automated (not fully), which freed the time for several employees. Currently being used by the finance department. The BOT requires minimum setup time and mapping. Around 1600 invoices have been uploaded since August 2021.

Tools: Python (standard library)

BACKGROUND/PROCEDURE:

Created a simple bot that uses an invoice mapping from csv file, and moves invoices from a repository folder into its respective program/project/subproject, and proper period (YYYY and MM MMM YYYY) directory. INVOICE PDF READER

Tools: Python (pandas, PyPDF2, tkinter, regex)

BACKGROUND/PROCEDURE:

Hundreds of invoices are processed monthly, and each of them requires to be renamed using company's SOP and its information transposed into excel. With PyPDF2, I was able to create a script which extracts all the strings from the PDF invoice, filters the information using RegEx, and appends them into an excel spreadsheet. This script helps ease out the manual data entry portion, as well as the invoice renaming process. This script also supports the INVOICE BOT UPLOADER.
